Για να τρέξετε το πρόγραμμα πρέπει να μεταφερθείτε μέσω Command Line στο directory 
που είναι αποθηκευμένα τα Missionaries_and_Cannibals.py και util.py αρχεία.
Στη συνέχεια εκτελώντας την εντολή 'python Missionaries_and_Cannibals.py N M K' το πρόγραμμα εκτελείται
(όπου Ν, Μ, Κ ακέραιοι αριθμοί και πιο συγκεκριμένα:
N ο αριθμός των κανιβάλων και των ιεραποστόλων στην αρχική κατάσταση
Μ η μέγιστη χωρητικότητα της βάρκας
και Κ ο μέγιστος επιτρεπόμενος αριθμός διασχίσεων του ποταμού)
